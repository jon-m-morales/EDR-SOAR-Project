<h2>Account Creation</h2>
<h3>This section details how to create a LimaCharlie account for access to the web portal.</h3>
<b>
1. Navigate to https://app.limacharlie.io/signup and setup a new account.
   <img width="632" height="578" alt="image" src="https://github.com/user-attachments/assets/a01bcffd-568c-4e27-b58a-7a70321e9813" /><br>
2. Fill in the requested information, verify your e-mail, and then answer the welcome questions and press the get started button. <br>
   <img width="415" height="233" alt="image" src="https://github.com/user-attachments/assets/d74842d9-5e4d-41f9-8719-abb354b6780e" /><br>
   <img width="762" height="837" alt="image" src="https://github.com/user-attachments/assets/a260656b-da01-4c04-b368-ca619a9dd7b7" /><br>
3. Create your organization by clicking the "Create Organization" button and then fill out the information. I opted to use the EDR template as that is my goal with for this service.<br>

<h2>Installation Key Setup</h2>
<h3>This section details how to create an installation key, to be used when installing sensors.</h3>
<b>
1. Now once it's completed creating the Organization and you've loaded the dashboard/home page, navigate to Sensors>Installation Keys and press the "Create Installation Key". <br>
<img width="2510" height="555" alt="image" src="https://github.com/user-attachments/assets/fee36d1a-baaa-4c02-a464-ca9b737b7bd3" />
2. Now name your key, and click create. Afterwards your new key should be visible in your list of keys. <br>
<img width="742" height="387" alt="image" src="https://github.com/user-attachments/assets/bc4e1408-fed5-4b09-abaf-8125ad1a8be7" />
</b>

<h2>Sensor Installation</h2>
<h3>This section details how to install LimaCharlie sensor on an endpoint.</h3>
<b>
1. Scroll down undernath your installation keys, and you will find the Sensor Downloads section. In the EDR category select the OS you plan to install on. In my example I will be installing on a Windows x64 bit.
